# Code of Conduct

This project welcomes thoughtful, respectful collaboration.

## Expected Behavior

- Be constructive.
- Critique ideas, not people.
- Explain tradeoffs.
- Assume good intent.
- Respect safety boundaries.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Hate speech
- Unsafe or deceptive product guidance
- Attempts to add impersonation, fraud, or abuse workflows

## Reporting

Open an issue or contact the maintainers if you see behavior that violates this code.
