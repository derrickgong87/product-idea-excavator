# Product Idea Excavator 中文说明

Product Idea Excavator 是一个中英双语的产品想法挖掘 Skill。

它适合：

- 创业者
- 产品经理
- 独立开发者
- AI 产品构建者
- 任何脑海中有产品想法但还没想清楚的人

它不会一上来就写 PRD，而是通过多轮主动提问，帮助你把产品 idea 挖掘清楚。

## 它解决什么问题？

很多产品想法一开始都是模糊的：

- “我想做一个 AI 工具。”
- “我想做一个语音 APP。”
- “我想做一个 SaaS。”
- “我想把这个想法写成 PRD。”

普通规划模式通常擅长把已知目标拆成步骤。这个 Skill 更关注目标本身是否清楚、是否值得做、该从哪里切。

## 它会追问什么？

- 第一批用户是谁？
- 有什么证据证明需求真实存在？
- 用户现在怎么解决这个问题？
- 现状方案让用户付出了什么成本？
- 最窄 MVP 是什么？
- 哪些功能不该做？
- 技术栈怎么选？
- AI 是否真的必要？
- 最大风险是什么？
- 什么时候可以总结 PRD？

## 中英双语如何工作？

你用中文，它就全程中文。

You write in English, it responds in English.

不会在同一次用户体验里混杂两种语言，除非你明确要求双语输出。

## 快速开始

把 `product-idea-excavator/` 复制到你的 Skills 目录，然后用类似提示开始：

```text
我有一个产品 idea，但还不成熟。请你像顶级产品经理一样，通过提问帮我把它挖掘清楚。
```

当你说：

```text
我确认没有其他补充了，可以总结 PRD。
```

它会自动输出结构化 PRD。
